# 🏸 羽毛球接龙考勤小程序

基于微信云开发的小程序，用于羽毛球群的接龙考勤管理。

## 功能

- **📝 活动接龙** — 发布活动预告，粘贴微信群接龙文本，自动解析微信号和昵称
- **📊 月度考勤** — 按月查看每个人的出勤次数，带考勤表和排名
- **👥 成员管理** — 参与人员自动累积，支持微信号识别

## 数据模型

### `activities` 集合
| 字段 | 说明 |
|------|------|
| `date` | 活动日期 |
| `title` | 活动标题 |
| `signups` | 参与人数组 `[{wxId, nickname}]` |

### `players` 集合
| 字段 | 说明 |
|------|------|
| `wxId` | 微信号（唯一标识） |
| `nickname` | 群昵称 |
| `aliases` | 曾用昵称列表 |

## 使用流程

1. 在群内发起接龙（或复制群里的接龙）
2. 打开小程序 → 粘贴接龙文本 → 自动解析并保存
3. 每月自动生成考勤表和排名

## 部署前配置

1. 在 `miniprogram/app.js` 中配置云环境 ID
2. 创建数据库集合：`activities`、`players`
3. 上传部署 `cloudfunctions/badminton` 云函数（云端安装依赖）
4. 设置云函数定时触发器（可选，用于每日自动统计）
